# Портфолио — структура папок

Кидай файл в нужную папку — на сайте он появится автоматически (Vite подхватит при следующей загрузке dev-сервера или при `npm run build`).

| Папка          | Что внутри                                              | На сайте показывается как |
|----------------|---------------------------------------------------------|---------------------------|
| `marketplace/` | Карточки WB / Ozon — главные слайды, инфографика       | **Карточка WB**           |
| `sku/`         | Линейки одного бренда / SKU в одном стиле               | **Линейка SKU**           |
| `neurophoto/`  | Нейрофотосессии товара (студийные кадры из плохих)      | **Нейрофото**             |
| `video/`       | Рекламные ролики (.mp4 / .webm)                         | **Видео**                 |
| `case/`        | Кейсы под товар (обогреватели, пылесосы, степперы и т.д.) | **Кейс**                |
| `social/`      | Посты для соцсетей, сторис, обложки                     | **Соцсети**               |
| `brand/`       | Бренд-визуалы, логотипы, фирменный стиль                | **Бренд**                 |
| `visual/`      | Свободные визуалы / арты / референсы                    | **Визуал**                |
| `slide/`       | Отдельные слайды-вставки (преимущества, факты)          | **Слайд**                 |

## Поддерживаемые форматы

- Картинки: `.png` `.jpg` `.jpeg` `.webp` `.avif`
- Видео: `.mp4` `.webm`

## Заголовок плитки

Берётся из имени файла (без расширения, `_` и `-` заменяются на пробелы).
Если хочешь красивый заголовок — назови файл по-человечески:
`степпер-главный-слайд.png` → «степпер главный слайд»

## Добавить новую категорию

1. Создать папку, например `src/portfolio/poligraphy/`
2. В `src/data/works.js` добавить строчку в `categoryMap`:
   ```js
   poligraphy: { label: "Полиграфия", order: 10 },
   ```
